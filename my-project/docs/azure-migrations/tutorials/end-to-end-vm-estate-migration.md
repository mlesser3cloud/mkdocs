# Tutorial: End-to-end VM estate migration

This tutorial describes a repeatable wave for VM-heavy estates.

## Steps

1. Select a wave and confirm owners.
2. Map dependencies and target network placement.
3. Validate sizing assumptions.
4. Replicate and perform test migrations.
5. Execute cutover with validation owners present.
6. Stabilize and hand over to operations.

## Outputs

- Updated inventory with Azure resource identifiers
- Runbooks updated based on lessons learned
- Post-wave review completed

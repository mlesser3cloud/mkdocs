# Acceptance criteria

Acceptance criteria should be objective, testable, and owned.

## Baseline (most workloads)

- Monitoring and alerting configured
- Logs flowing to the central destination
- Backup configured and at least one restore tested
- Access model enforced (least privilege and audited)
- Functional smoke tests pass

## Add for critical workloads

- Performance meets baseline under expected load
- DR plan exists and is rehearsed
- Vulnerability management SLAs defined and tracked
